방법 2: 수동 복사

아래 코드를 복사하여 북마크 URL에 붙여넣으세요:

javascript:(function(){const baseUrl=window.location.origin+window.location.pathname.replace(/[^\/]*$/,'');const cssLink=document.createElement('link');cssLink.rel='stylesheet';cssLink.type='text/css';cssLink.href=baseUrl+'ui.css';document.head.appendChild(cssLink);window.ZuppSEO=window.ZuppSEO||{};window.ZuppSEO.baseUrl=baseUrl;const scripts=['zupp.js','analyzers.js','analyzers-extended.js','analyzers-technical.js','analyzers-geo-mobile.js','ui.js'];let loaded=0;function loadScript(src){const script=document.createElement('script');script.src=baseUrl+src;script.onload=()=>{loaded++;if(loaded===scripts.length){setTimeout(()=>{if(window.ZuppSEO&&window.ZuppSEO.run){window.ZuppSEO.run();}},100);}};script.onerror=()=>{console.error('Failed to load '+src);};document.head.appendChild(script);}scripts.forEach(loadScript);})();

방법 3: 즉시 테스트

북마클릿을 설치하지 않고 바로 테스트해보세요:

📌 설치 방법

  1. 북마크 바 표시: Ctrl+Shift+B (Windows) 또는 Cmd+Shift+B (Mac)
  2. 드래그 방법: 위의 보라색 버튼을 북마크 바로 드래그
  3. 수동 방법:
    • 북마크 바에서 우클릭 → "페이지 추가"
    • 이름: zupp SEO 분석
    • URL: 위의 코드 복사 후 붙여넣기

⚠️ 문제 해결

  • 아무 일도 일어나지 않음: 개발자 콘솔(F12)을 열고 에러 메시지 확인
  • 파일을 찾을 수 없음: 모든 JS 파일이 같은 폴더에 있는지 확인
  • 권한 오류: 로컬 파일에서는 작동하지 않을 수 있음. HTTP 서버 필요
