분석 시작!

분석하고 싶은 웹사이트에서
북마크에 추가된 "zupp SEO 분석"을
클릭하면 즉시 분석이 시작됩니다

📌 드래그가 안 되는 경우 (수동 설치)
  1. 북마크 바에서 우클릭 → "페이지 추가" 또는 "북마크 추가" 선택
  2. 이름: zupp SEO 분석
  3. URL: 아래 코드를 복사해서 붙여넣기
    javascript:(function(){const BASE_URL='http://localhost:8000/';const cssLink=document.createElement('link');cssLink.rel='stylesheet';cssLink.type='text/css';cssLink.href=BASE_URL+'ui.css';document.head.appendChild(cssLink);window.ZuppSEO=window.ZuppSEO||{};window.ZuppSEO.baseUrl=BASE_URL;const scripts=['zupp.js','analyzers.js','analyzers-extended.js','analyzers-technical.js','analyzers-geo-mobile.js','ui.js'];let index=0;function loadNextScript(){if(index>=scripts.length){setTimeout(()=>{if(window.ZuppSEO&&window.ZuppSEO.run){window.ZuppSEO.run();}},100);return;}const src=scripts[index];const script=document.createElement('script');script.src=BASE_URL+src;script.onload=()=>{index++;loadNextScript();};document.head.appendChild(script);}loadNextScript();})();
  4. 저장 클릭 → 완료! 🎉
