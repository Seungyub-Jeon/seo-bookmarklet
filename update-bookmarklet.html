🚀 ZuppSEO 북마클릿 업데이트

⚠️ 중요: 성능 탭과 모바일 탭이 제거된 새 버전입니다.
기존 북마클릿을 삭제하고 아래의 새 북마클릿으로 교체해주세요.

📋 업데이트 방법

  1. 기존 북마클릿 삭제: 북마크바에서 기존 "ZuppSEO" 북마클릿을 우클릭 → 삭제
  2. 새 북마클릿 추가: 아래 보라색 버튼을 북마크바로 드래그
  3. 또는: 버튼을 우클릭 → "링크를 북마크에 추가"

아래 버튼을 북마크바로 드래그하세요:

✨ ZuppSEO v2.0
✅ 업데이트 내용:
  • 성능 탭 제거됨
  • 모바일 탭 제거됨
  • analyzers-geo-mobile.js → analyzers-geo.js로 변경
  • AI 검색 최적화(GEO) 기능은 유지

🧪 테스트 방법

  1. 북마클릿 업데이트 완료 후
  2. 아무 웹사이트 방문
  3. 북마크바에서 "ZuppSEO v2.0" 클릭
  4. 성능 탭과 모바일 탭이 없어진 것 확인

🔧 문제 해결

만약 여전히 오류가 발생한다면:

  1. 브라우저 캐시 삭제 (Ctrl/Cmd + Shift + Delete)
  2. 브라우저 재시작
  3. 북마클릿 다시 추가
