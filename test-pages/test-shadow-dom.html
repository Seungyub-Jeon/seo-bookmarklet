Shadow DOM 테스트 페이지

이 페이지는 CSS 충돌을 시뮬레이션합니다.

theninetosix.com과 같이 공격적인 CSS가 적용되어 있습니다.

테스트 시나리오:

  • nav 태그에 position: fixed 적용됨
  • header 태그에 강제 스타일 적용됨
  • 모든 div와 button에 글로벌 스타일 적용됨
  • .zupp-header 클래스에 직접 스타일 적용됨

예상 결과:

  • Shadow DOM을 사용한 헤더/네비게이션은 영향받지 않음
  • 나머지 콘텐츠 영역은 일부 영향받을 수 있음
